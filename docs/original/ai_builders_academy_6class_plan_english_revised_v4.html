AI Builders Academy

A hands-on AI course where high school students learn ChatGPT, prompt engineering, VS Code, Codex, web UI building, game development, MCP, YouTube integration, Browser Use, Computer Use, OpenClaw, OAuth, and AI agents through practical projects.

Course Vision

Main Goal

Students will move from simply “using AI” to building with AI. They will create real projects such as web UIs, games, research agents, and automation workflows.

Class-by-Class Lesson Plan

CLASS 1

Meet Your AI Teammate — ChatGPT Basics

Students learn that ChatGPT is not just a search engine. It can act as a brainstorming partner, tutor, coach, writer, and project assistant. Students practice the prompt formula: Role + Task + Context + Format.

HookCan AI do my homework?
DemoBad prompt vs. good prompt comparison
HomeworkCreate an AI assistant with 5 example conversations
CLASS 2

AI Coding Environment — VS Code + Codex Setup with Web UI Example

This class combines environment setup with a practical web UI example. Students install or open VS Code, set up Codex, learn the basic user interface, understand folders and files, practice token management, and use Codex to create a simple web UI. The web UI is used as the example project for learning the tool.

HookCan AI become your coding partner inside VS Code?
DemoSet up Codex, open a folder, generate a simple web UI, and preview it
HomeworkComplete setup and customize the web UI with one personal section
CLASS 3

AI Game Studio — Build a Game with Codex

Students use the VS Code + Codex environment prepared in Class 2 to build a simple browser game. Options include a clicker game, dodge game, snake-style game, or simple space shooter. The focus is on asking Codex to generate, explain, debug, and improve the game step by step.

HookCan AI build a game that your friends can play?
DemoUse Codex to create a playable browser game and add score or levels
HomeworkAdd one new feature and get feedback from a parent or friend
CLASS 4

AI Agents & MCP — YouTube Research Agent

Students learn the concept of an AI agent and MCP. MCP is introduced as a simple idea: a connector that lets AI use external tools. Students build or simulate a YouTube Research Agent that finds videos, summarizes them, and compares results.

HookCan AI use internet tools by itself?
DemoFind and summarize YouTube videos about a student-selected topic
HomeworkCreate a Top 3 video comparison report
CLASS 5

Browser Use & Computer Use — AI as an Operator

Students learn how AI can use a browser or computer-like interface to perform tasks. The class focuses on safe, understandable demos: opening pages, extracting information, filling simple forms, comparing results, and explaining what the agent is doing.

HookCan AI operate a browser like a human assistant?
DemoBrowser Use and Computer Use workflow with a simple task
HomeworkDesign a 5-step browser task that an AI agent could perform
CLASS 6

OpenClaw + Codex OAuth — Building an AI Agent Workflow

Students are introduced to OpenClaw as a more advanced AI agent framework. The class explains OAuth in simple terms, shows how Codex OAuth connects tools securely, and demonstrates an OpenClaw-style workflow. Students finish by presenting their final AI project concept or demo.

HookCan we connect AI tools like apps on a phone?
DemoOpenClaw workflow, OAuth login concept, and agent connection
FinalFinal AI project presentation and peer feedback

Class Format

1. Hook

Each class starts with a question or demo that makes students curious immediately.

  • Can AI become my coding partner?
  • Can AI build a game?
  • Can AI use YouTube or a browser?

2. Live Demo

The instructor shows a working example before explaining details. Students see the result first.

  • Type a prompt
  • Generate or modify a project
  • Debug and improve live

3. Hands-on Build

Students create their own version, make mistakes, ask AI for help, and improve through iteration.

  • Individual project
  • Peer testing
  • Short presentation

Assignments and Assessment

Assessment is based on projects, not tests.

Each class includes a small assignment that can be completed in about 30–60 minutes. The assignments are designed to be shareable with parents, friends, or classmates.

  • Class 1: AI Assistant with example conversations
  • Class 2: VS Code + Codex setup screenshots and customized web UI example
  • Class 3: Playable browser game built with VS Code + Codex
  • Class 4: MCP + YouTube Research Agent report
  • Class 5: Browser/Computer Use task design
  • Class 6: OpenClaw-based final project concept or demo presentation
