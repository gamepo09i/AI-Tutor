4-Lesson High School AI Creator Course

AI Builders Academy

A hands-on AI course where high school students learn ChatGPT, prompt engineering, VS Code, Codex, web UI building, game development, MCP, YouTube integration, Browser Use, Computer Use, OpenClaw, OAuth, and AI agents through practical projects.

AudienceHigh School Students
Duration4 Weeks
ScheduleOnce a Week / 90 min
FormatDemo + Hands-on Project

Course Vision

Main Goal

Students will move from simply “using AI” to building with AI. They will create real projects such as web UIs, games, research agents, and automation workflows.

Practical AI Creator Mindset Build First Fun Projects

Teaching Philosophy

This course avoids heavy theory, advanced math, and academic machine learning lectures. The focus is on curiosity, creativity, and tools students can actually use.

Lesson-by-Lesson Plan

LESSON 1

AI Teammate Foundations — ChatGPT, Prompting, and First Assistant

Students learn that AI is a creative teammate, not just a search box. They practice Role + Task + Context + Format, test answers, and build a personal assistant they can improve.

HookCan AI do my homework?
90 minDemo, assistant build, pair test, revise, exit ticket
HomeworkCreate an assistant with 5 conversations and one safety rule
LESSON 2

Codex Build Studio — VS Code, Web UI, and Space Invaders Basics

Students set up the coding workspace, learn files and previews, build a simple web UI, then use Codex to create the basics of a Space Invaders-style browser game with player movement, shooting, enemies, score, lives, and restart.

HookCan AI become your coding partner inside VS Code?
Final InteractiveBuild a basic Space Invaders game with Codex help
HomeworkExpand and refine the game to make it more fun
LESSON 3

AI Agents and Research Workflows — MCP, YouTube, Browser Safety

Students learn how agents plan steps, use tools, and need human supervision. MCP, YouTube research, Browser Use, Computer Use, verification, and safety rules are combined into one workflow lesson.

HookCan AI use internet tools by itself?
DemoCompare three beginner-friendly YouTube videos
HomeworkCreate a video report and safe browser task design
LESSON 4

Final Agent Project — OpenClaw, OAuth, Demo, and Next Build

Students connect prompting, coding, games, agents, MCP, browser safety, OpenClaw, and OAuth into a final project concept or demo they can explain. They finish with peer feedback and a next-build plan.

HookCan we connect AI tools like apps on a phone?
DemoOpenClaw-style workflow and OAuth permission concept
Final2-minute project presentation and peer feedback

Lesson Format

1. Hook

Each 90-minute lesson starts with a question or demo that makes students curious immediately.

  • Can AI become my coding partner?
  • Can AI build a game?
  • Can AI use YouTube or a browser?

2. Live Demo

The instructor shows a working example before explaining details. Students see the result first.

  • Type a prompt
  • Generate or modify a project
  • Debug and improve live

3. Hands-on Build

Students create their own version, make mistakes, ask AI for help, and improve through iteration.

  • Individual project
  • Peer testing
  • Short presentation

Assignments and Assessment

Assessment is based on projects, not tests.

Each lesson includes a small assignment that can be completed in about 30–60 minutes. The assignments are designed to be shareable with parents, friends, or classmates.

  • Lesson 1: AI Assistant with example conversations and one safety rule
  • Lesson 2: Customized web UI plus Space Invaders starter; homework expands and refines the game
  • Lesson 3: MCP + YouTube research report and safe browser task design
  • Lesson 4: Final project concept, demo, or 2-minute presentation
